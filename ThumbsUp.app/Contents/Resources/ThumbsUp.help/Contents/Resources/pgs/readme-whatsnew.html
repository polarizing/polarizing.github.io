Version 4.5.2

This maintenance release speeds up thumbnail creation, retains aspect ratio, and fixes minor issues.

  • NEW: Created icons retain aspect ratio.
  • Improved: Thumbails are created up to 50 percent faster.
  • Fixed: Thumbnails were twice as large as supposed to on Retina displays.
  • Fixed: Glitch of Limit and Scale fields on OS X Mavericks.

Version 4.5.1

This maintenance release adds 64-bit and Gatekeeper support, revises the application menu, improves the overall compatibility, and fixes minor issues.

  • NEW: 64-bit support.
  • NEW: Support for Gatekeeper.
  • NEW: Help menu and built-in help.
  • Improved: Slightly revised application menu.
  • Improved: Overall compatibility.
  • Fixed: Orientation of created thumbnails wasn't always correct (e.g. for pictures taken in portrait mode).
  • Fixed: Thumbnailing of movies crashed the app.

Note: Starting with this version ThumbsUp requires an Intel-based and Mac OS X 10.6.8 or later.

Version 4.5

  • NEW: Transformations (rotate left/right, flip horizontal/vertical) defined by TIFF/EXIF information and used by older versions of Preview supported.
  • Improved: Up to two times faster thumbnailing of images.
  • Improved: Better and faster thumbnailing of QuickTime movies.
  • Fixed: Glitch of thumbnailing of certain image types, e.g. PDF documents.
  • Fixed: Minor packaging glitches.

Note: Starting with this version ThumbsUp requires Mac OS X 10.5 or later.

Version 4.4

  • Improved: Created icons are optimized for Mac OS X 10.4 or later.
  • Improved: Thumbnailing of QuickTime movies uses the middle of the movie instead of the beginning.
  • Improved: Optimized for Mac OS X 10.4 ‘Tiger’ and 10.5 ‘Leopard’.
  • Improved: Memory usage and startup performance.
  • Removed: Support for Mac OS X 10.3.9 Panther.

Version 4.3

  • NEW: Universal binary for Intel- and PowerPC-based Macs.
  • NEW: French localization.
  • NEW: Support for QuickTime file types added by third-party components (Mac OS X 10.4.x or later).
  • Fixed: Minor glitches.

Note: Starting with this version ThumbsUp requires Mac OS X 10.3.9 or later.
