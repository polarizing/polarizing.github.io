    Installing, updating, uninstalling
 

Installing and updating

Copy ThumbsUp into the Applications folder or any other directory you have access to. As ThumbsUp requires drag-and-drop, it's also a good idea to add it to the Dock or create an alias for it on the Desktop.

To update an existing version of ThumbsUp, simply copy the application package from the disk image to your Applications folder, replacing the old version.

Uninstalling

To remove (uninstall) ThumbsUp completely from your Mac, simply move the ThumbsUp application package to the trash.
