Keyboard symbols:

  • ⌘ The Command key, usually next to the Space bar.
  • ⌥ The Option key.
  • ⌃ The Control key (usually labeled with "ctrl").
  • ⇧ The Shift key.
  • ⇥ The Tab key.
  • ↩ The Return key.

Frequently used terms:

  • Shortcut: A combination of keys on the keyboard pressed together to start an action.
  • Moving: Changing a window's position on the screen.
  • Resizing: Changing a window's size.
