Appendix

 

Thank you for using our freeware. ThumbsUp is a simple, drag-and-drop based utility to create thumbnails or icons for a bunch of pictures or movies and supports all image and movie formats supported by OS X and QuickTime (including PDF and PostScript documents).

Thumbnails are either scaled by percentage or limited to a maximal size and created as JPEG, JPEG 2000, TIFF, PNG, BMP or GIF files. In addition, it is possible to specify antialiasing, sharpening, and compression to optimize the results. Use ThumbsUp to convert PDF screenshots easily to other image formats using drag-and-drop and scaling to 100 percent. [Read more...]
